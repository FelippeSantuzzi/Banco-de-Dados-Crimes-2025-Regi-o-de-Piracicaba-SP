# Análise de Criminalidade — Região de Piracicaba/SP

Projeto da disciplina de Mineração de Dados aplicado sobre dados reais de criminalidade 
de 52 municípios da região de Piracicaba/SP (SSP-SP, 2025). Utilizamos K-Means para 
agrupar cidades por perfil criminal e Apriori para identificar associações entre tipos de crime.
